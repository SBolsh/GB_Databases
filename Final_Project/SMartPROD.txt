SMartPROD

SMartPROD - это система которая помогает работать с новыми функциями (feature requests)
в продуктах. Система поддерживает пользователей на нескольких этапах работы:

- Идея (idea) - описание новой функции в свободной форме
- Запрос функции (feature request) - уже более формальное описание, привязанное 
к конкретному продукту и конкретному бизнес-процессе, который эта фнукция должна 
улучшить.
- Роадмап (Roadmap) - наборы feature request которые уже стоят в планах для реализации 
отдельными командами.

Помимо работы внутри компании-разработчика с жизненным циклом идей/запросов на фнукции,
система позволяет клинетам компании активно участвовать в процессе (видеть статутсы, 
отправлять сообщения).

Особое место уделяется тому, чтобы новый функции разрабатывались именно для решения
задач клинетов, а не "просто потому что команда так хочет". Для этого отдельно отмечаются
те идеи которые относятся к задачам клинетов (cusotmer idea), для того чтобы перевести
идею в запрос на создание новой функции обязательно нужно указать какой процесс в работе
компании-клинета она будет автоматизировать/улучшать/изменять, для того чтобы взять запрос
на создание новой фукнции в работу, обязательно нужно указать какой конертной компании эта
новая функция будет полезна.
